
# 字面常量

- 无类型；
- 只要该常量在相应类型的值域范围内，就可以作为该类型的常量；

# 字符类型

 Go 语言支持两种字符类型：

 - byte : uinit8 的别名，代表 UTF-8 字符串的单个子节的值；
 - rune : 代表单个 Unicode 字符；

# 布尔类型

使用关键字 bool ，可以赋值为预定义的 true 和 false ；

> 布尔类型不能接受其它类型的赋值，不支持__自动__或__强制__的类型转换；

# 整型

注意：

- int 和 int32 在 Go 语言中被认为是两种不同类型，并且编译器不会帮你自动做类型转换；
- int 和 uint 的长度是平台相关的，不能对这些类型的长度做任何假设；
